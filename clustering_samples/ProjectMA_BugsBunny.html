Abstract

Below dataset covers information about gender,student performance (math, reading, writing); etnicity; level of education; lunch(free/reduced); test-preparation course. Dataset is retrieved from kaggle[1].Classification andlogistic regression methods are investigated in this study.

Decision tree is going to be used as classification method in the first model. In the second model, gender of students are tried to be predicted with logistic regression.Study will start with an explanatory data analysis. Pre-processing part will follow it. Finally, results are going to be discussed for both models, separately.

Model- Result & Discussion

In the first model, gender classification is done. Training and testing data are taken as 80% and 20%, respectively.

Equation

In below equation, max dept is set to 4, after trial& errors.Overfitting issue is avoided.

m <- rpart(gender~., data= original_train, method="class", maxdepth=4)

  • All the features are feed into data, and classification is done based exam scores.

  • In the first model, it is seen that, writing score with threshold 70, is the most deterministic
    factor. All population is distributed by 48%-52% based on this.

  • Math score is the second important discriminative factor.

  • Accuracy difference between test and training data set is around 3%, so there is no issue on overfitting.

  • Model can be evaluated as good enough, with 79% accuracy on test set.

Main Take Aways

Logistic regression gave higher accuracy score.The reason would be the fact that, dataset has good distribution, which is quite close to gaussion. If the dataset was larger than 1000 samples, accuracy of decision tree would be higher.The reason is decision tree is very expensive in terms of sample size.Therefore it requires higher amount of data size for higher accuracy.
